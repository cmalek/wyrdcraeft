Wright's paradigms (Old English Grammar, 1908)
==============================================

The morphology generator's **wright** column (and paradigm labels) use the
example words and numbering from Joseph Wright and Elizabeth Mary Wright, *Old
English Grammar* (1908; 1914 reprint). The generator uses these paradigm names
as keys for inflectional patterns. This page lists the main paradigms from
Wright's grammar and notes which ones the wyrdcraeft generator implements.

Nouns
-----

Wright's noun paradigms are organized by stem type and gender. The following
are the principal example words (lemmas) used in the grammar:

.. list-table::
   :header-rows: 1
   :widths: 15 55

   * - Paradigm
     - Description
   * - *stán*
     - Masculine a-stem (long stem). “Stone.”
   * - *dæg*
     - Masculine a-stem (short stem, vowel variation). “Day.”
   * - *cynn*
     - Neuter ja-stem. “Kin, race.”
   * - *word*
     - Neuter a-stem (long stem). “Word.”
   * - *hof*
     - Neuter a-stem (short stem). “Court, dwelling.”
   * - *fæt*
     - Neuter a-stem (short stem, syncope). “Vessel.”
   * - *ār*
     - Feminine ō-stem. “Honour, property.”
   * - *strengu*
     - Feminine ō-stem (abstract in *-u*). “Strength.”
   * - *feld*
     - Masculine a-stem (syncope). “Field.”
   * - *hand* / *duru*
     - Feminine consonantal (n- and r-stems). “Hand,” “door.”
   * - *sunu*
     - Masculine u-stem. “Son.”
   * - *bearu*
     - Masculine wa-stem. “Grove.”
   * - *bealu*
     - Neuter wa-stem. “Evil, harm.”
   * - *beadu*
     - Feminine in *-u*. “Battle.”
   * - *guma*
     - Masculine n-stem (weak). “Man.”
   * - *fréa*
     - Masculine n-stem (weak). “Lord.”
   * - *tunge*
     - Feminine n-stem (weak). “Tongue.”
   * - *béo*
     - Feminine n-stem (weak). “Bee.”
   * - *éage*
     - Neuter n-stem (weak). “Eye.”
   * - *wígend*
     - Masculine nd-stem (present participle noun). “Fighter, warrior.”

Adjectives
----------

.. list-table::
   :header-rows: 1
   :widths: 15 55

   * - Paradigm
     - Description
   * - *blinda*
     - Weak adjective declension. “Blind.”
   * - *glæd* / *til*
     - Strong adjective (æ/ea-stem type). “Glad,” “good.”
   * - *blind*
     - Strong adjective (a-stem). “Blind.”
   * - *hēah* / *þweorh*
     - Strong adjective (h-stem). “High,” “perverse.”
   * - *manig*
     - Strong adjective (i-stem type). “Many.”
   * - *hālig*
     - Strong adjective (ja-stem with syncope). “Holy.”
   * - *wilde*
     - Strong adjective (i-stem, stem drops final *-e*). “Wild.”
   * - *gearu*
     - Strong adjective (u-stem). “Ready.”

Verbs
-----

(1–3), preterite-present, and anomalous verbs. The following are the example

verbal paradigms file:

Generator coverage
------------------

The wyrdcraeft morphology generator **covers** the following:

- **Nouns:** The generator implements inflection for the noun paradigms *stán*,
  *cynn*, *word*, *hof*, *dæg*, *fæt*, *ār*, *strengu*, *feld*, *hand*, *duru*,
  *sunu*, *bearu*, *bealu*, *beadu*, *guma*, *fréa*, *tunge*, *béo*, *éage*,
  *wígend*. Assignment is based on the dictionary’s Wright section codes and
  heuristics; unassigned nouns default to *stán* (masc./uncertain), *ār*
  (fem.), or *word* / *hof* (neut. by stem length). Optional r-stem noun
  generation can be enabled separately.

- **Adjectives:** The generator implements weak (*blinda*) and the strong
  types *glæd*/*til*, *blind*, *hēah*/*þweorh*, *manig*, *hālig*, *wilde*,
  *gearu*. Assignment uses the dictionary’s Wright codes and stem-based
  heuristics.

- **Verbs:** The generator implements all verb paradigms defined in the
  packaged verbal paradigms file (``para_vb.txt``). Each entry has a paradigm
  title (e.g. *bindan*, *helpan*, *déman*) that corresponds to a Wright-style
  example verb. The **wright** column in the TSV output is the paradigm name
  (verb title or noun/adjective paradigm) assigned to that form.

**Not covered**

The following Wright paradigms are **not** implemented in the generator (they
do not appear in ``para_vb.txt``):

- **Verbs:** *sittan* (Strong V, “sit”), *macian* (Weak class 2, “make”),
  *hyngrian* (Weak class 2, “hunger”).

Nouns and adjectives listed above are fully covered; no Wright noun or
adjective paradigm from this page is missing.
